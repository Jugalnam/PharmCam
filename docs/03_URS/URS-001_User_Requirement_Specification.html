URS-001

User Requirement Specification — 사용자 요구사항 명세

PharmCam — 검체·시료 사진 기록 시스템
확정 v1.0 요구사항 42 RA 추적 연결
태그필수 코어, 끌 수 없음
 설정가능 회사별 on/off·값
 조건부 모듈 사용 시 필수
출처 각 요구사항은 RA 위험 ID에서 도출

이 문서가 하는 일: "시스템이 반드시 해야 할 일"을 요구사항으로 정의한다. 각 요구사항은 RA의 위험 통제에서 도출되며(추적성), DQ·OQ/PQ로 연결된다. 엄격한 GMP 코어 + 회사별 설정 유연성 원칙을 따른다.
